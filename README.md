# 概要
[こちら](https://zenn.dev/septeni_japan/articles/2024-12_kintone_iac)で紹介しているKintone管理システムのコードです。

# 前提
- Node.js >= 22.9.0
- npm >= 10.8.3

# Install Dependencies
```bash
npm install
```

# Command
### donwload
```bash
npm run download -- -t <アプリのディレクトリ名>
```

### diff
```bash
npm run diff -- -t <アプリのディレクトリ名>
```

### deploy
```bash
npm run deploy -- -t <アプリのディレクトリ名>
```
